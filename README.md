# xza (lodash mixin)

これは簡単なコードの寄せ集めです。
読み込むだけでlodashを拡張します。

```
npm install xza --save
```

```
_ = require('xza').lodash
```

## utility

- \_.sizeof( obj )
- \_.strToNumber( str, returnValue = NaN )
- \_.replaceAll( str, obj )
- \_.toHalfString( str )
- \_.whiteSpaceRemover( str )
- \_.toDateString( obj )
